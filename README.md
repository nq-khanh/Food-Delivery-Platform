## Mô tả

HKT là nền tảng đặt và giao đồ ăn trực tuyến được xây dựng trong khuôn khổ môn học **Phát Triển Hệ Thống Web**.  
Hệ thống cho phép người dùng tìm kiếm món ăn, đặt hàng, theo dõi đơn hàng và hỗ trợ quản lý dành cho nhà hàng cũng như quản trị viên.
